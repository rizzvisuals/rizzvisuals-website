RIZZVISUALS WEBSITE — IPAD SETUP

UPLOAD TO GITHUB
Upload index.html, README.txt, and the assets folder. Do not upload only the ZIP.

REPLACE PLACEHOLDERS
1. Replace SMUGMUG_SHOP_URL in index.html with your SmugMug shop/gallery link.
2. Replace YOUR_EMAIL@example.com with your real email.

CLOUDFLARE PAGES
Connect the GitHub repository in Workers & Pages. Leave build command blank and use / as the output directory. Then add rizzvisuals.com under Custom domains.
